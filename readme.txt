== Description ==

Este plugin vai Adiciona campos personalizados (EAN, NCM, GTIN) aos produtos do WooCommerce.

Nenhuma configuração é necessária.

= Por que usar esse plugin? =

Esse plugin funciona para adicionar atributos ao seu produto que poderá ser utilizado em planilhas ou em sistemas externos.
